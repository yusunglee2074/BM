## Description

BM backend

## Prerequisites

- Node.js (version 16 or later)
- Docker (latest)

## Installation

```bash
$ docker-compose up -d
$ yarn install
```

## Running the app

```bash
# development
$ yarn start:dev

# production mode
$ yarn build && yarn start
```
